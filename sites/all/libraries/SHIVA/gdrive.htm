This utility will create a link to a document stored in Google Drive that can be used in SHIVA, MapScholar, or Qmedia:
  1. Go to your Google drive page and make sure the document is shared so that anyone with the link can view it (found in the Share menu after right-clicking).
  2. Copy the link to share from the highlighted text box to your clipboard.
  3. Paste the share link to the first text box and press ENTER.
  4. Copy the new link from the second text box.
